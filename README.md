# Student Management System

A simple command-line Student Management System written in Python.

## Features

* Add a new student
* Display all students
* Search students by name
* Delete a student with confirmation
* Input validation for user selections

## Technologies

* Python 3
* Lists
* Dictionaries
* Loops
* Conditional Statements
* Exception Handling (`try` / `except`)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/AidaHaghdadi/Student-Management-System.git
```

2. Navigate to the project directory:

```bash
cd Student-Management-System
```

3. Run the program:

```bash
python main.py
```

> If your main file has a different name, replace `main.py` with the correct filename.
